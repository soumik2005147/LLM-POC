Key Features

Web Search

Access current information from the web using Google Search API integration.

AI Processing

Leverage AI Pipe proxy for advanced data transformations and analysis workflows.

Code Execution

Execute JavaScript code for calculations, data processing, and complex logic.

Architecture

This agent is built using modern web technologies and follows the apiagent design pattern:

  • Frontend: HTML, CSS (Bootstrap 5.3), JavaScript ES6+ modules
  • LLM Integration: bootstrap-llm-provider for model selection
  • Error Handling: bootstrap-alert for user notifications
  • Templating: lit-html for reactive UI updates
  • Streaming: asyncLLM for real-time response streaming
  • Markdown: marked for rich text formatting

Getting Started

  1. Configure your LLM provider (OpenAI, Claude, etc.) in the main interface
  2. Enter your API key - it's stored locally in your browser
  3. Ask questions that require web search, calculations, or data processing
  4. Watch the agent reason through the problem step by step
  5. See real-time tool usage and results
